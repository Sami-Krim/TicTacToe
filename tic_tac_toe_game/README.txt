Ce fichier est destiné pour expliquer comment lancer ce projet et quelques informations générales.

Ce projet est un projet de Tic Tac Toe developé pour un mini projet du module de l'Intelligence Artificielle
Ce projet à été developé en utilisant l'engin des jeux Godot et programmé avec le langage GdScript (langage similaire a Python 
fonctionnant directement dans Godot)

Pour lancer le projet, on doit d'abord avoir l'engin des jeux Godot 4.0 
(télécharger ici: https://godotengine.org/download/windows/)
importer le projet dans Godot (dans la liste des projets, cliquez sur Import et naviguez vers le chemin du projet et selectionnez le fichier project.godot)
au lancement du projet, cliquer sur F5 pour lancer le projet.

les fichiers du dossier code sont des scriptes du jeux qui définient le fonctionnement général du jeu et la communication entre les differents elements du jeu
le fichier AI.gd concerne le code de l'IA du jeu.
L'adversaire du jeu a été codé en utilisant l'algorithme de recherche MinMax.
